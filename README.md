# evankoza.github.io

My personal portfolio — a single, hand-built site with **no framework and no build step**: just vanilla HTML, CSS, and JavaScript.

**Live:** https://evankoza.com

## Demos

Interactive experiments live in [`/experiments`](experiments):

- **Invoice Rush** — a receiving-desk reconciliation game (the playable companion to my invoicing-automation tool).
- **Lissajous Toy** — click any letter to reshape the harmonic curve that draws it.
- **Portrait / ASCII Tuner** — the live control panel behind the homepage hero.

## Stack

Zero dependencies. Everything is hand-written and rendered with the Canvas API.
